# 📰 AI-Powered News Article Analyzer

This is a Gradio-powered web app that analyzes any news article URL and extracts:

- 🧠 Title  
- ✍️ Authors  
- 📅 Publication Date  
- 📝 Summary  
- 💬 Sentiment (using TextBlob)  
- 🏷️ Top 10 Keywords
